App description

It is in react native expo in which users can add recipes, update, delete them. After making it work offline, I add online features in it such as signup/signin, online database. Users are able to store their recipes online bounded with their account.Registering or logging in automatically uploads all the offline recipes into online database. I added another feature of storing recipes offline until user connects to internet. This happens only if the user is logged into an account. I connected database with expressJS. The other featues includes a shopping cart where users can enter items with their prices. There is also a food randomizer where all the recipes are shuffled and one is shown. You may remove/add any recipe from the randomizer. I also added the ability to share recipes and add them into favourites. I used mongodb for database. The whole took me about a month or two as i was learning react native and backend along building it.
